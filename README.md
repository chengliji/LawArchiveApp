# 律所案卷归档小工具

[![Python](https://img.shields.io/badge/Python-100%25-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

一个为律所（律师）设计的案卷归档管理工具，帮助快速、高效地组织和管理法律案卷文件。

## 功能特性

- 📁 **案卷管理** - 快速创建和管理案卷档案
- 🔍 **智能搜索** - 快速查找相关案卷
- 📊 **数据统计** - 案卷统计和分析
- 💾 **批量操作** - 支持批量导入导出
- 🔐 **安全存储** - 案卷信息安全保护

## 快速开始

### 环境要求

- Python 3.7+

### 安装

```bash
# 克隆仓库
git clone https://github.com/chengliji/LawArchiveApp.git
cd LawArchiveApp

# 安装依赖
pip install -r requirements.txt
```

### 使用

```bash
python main.py
```
